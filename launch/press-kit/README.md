# Press kit

Self-service materials for journalists, KOLs, and partners.

## Contents

- `logo.svg` — anyfn wordmark, dark + light variants.
- `app-icon.svg` — the `fn(·)` mark.
- `og-image.png` — 1200×630 social preview.
- `demo.gif` — placeholder until you record the real 60-second demo.
- `screenshots/` — phone screenshots from each major screen (placeholder PNGs included; replace with real ones after the first build).
- `one-pager.md` — single-page company / project summary.

## Usage

Free to use under MIT with attribution. Don't recolour the logo. Don't apply effects (drop shadows, gradients) to the wordmark.

## Quoting

Founders are happy to be quoted as: "the anyfn team". Individual quote attribution by request via `press@anyfn.dev`.

## High-resolution video

Drop the master demo file at `launch/press-kit/demo-1080p.mp4` after recording. Provide a YouTube unlisted link to anyone who asks.
